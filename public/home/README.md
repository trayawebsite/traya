# Home page photography

Drop real photos here with these exact names and they appear automatically — a
refined warm placeholder shows until then (never a broken image). No code change
needed. Style: warm, natural light, shallow depth of field; real product / origin,
never stock-blue, lab, or clip-art.

| File                     | Where it shows      | Subject                                              | Aspect / size        |
| ------------------------ | ------------------- | ---------------------------------------------------- | -------------------- |
| `hero.jpg`               | Hero (right)        | Signature shot: ingredients in hand, an origin scene, or the founder at work | portrait, ≥1400×1680 |
| `group-alliums.jpg`      | Product tile        | Dehydrated onion / garlic flakes or powder           | 4:5, ≥800×1000       |
| `group-powders.jpg`      | Product tile        | Spray-dried fruit/veg powder (mango, tomato…)        | 4:5                  |
| `group-spices.jpg`       | Product tile        | Turmeric / chilli / cumin                            | 4:5                  |
| `group-herbs.jpg`        | Product tile        | Oregano / basil / dried culinary herbs               | 4:5                  |
| `group-nutraceutical.jpg`| Product tile        | Moringa / spirulina / amla                           | 4:5                  |
| `group-wellness.jpg`     | Product tile        | A2 ghee / cold-pressed oil / millets                 | 4:5                  |

- JPG or WebP; `next/image` optimizes/resizes automatically, so high-res originals are fine.
- The **founder portrait** is wired separately in `lib/site-settings.ts` (`founderPhoto`),
  currently the WordPress URL — self-host it here too (e.g. `founder.jpg`) when convenient.
- Tiles crop to 4:5 and the hero crops to portrait on desktop, so keep the subject centred.
